You should be registred Yandex user and add your site in Yandex Webmaster Service.== Installation ==1. Upload '/yandex-webmaster/' to the '/wp-content/plugins/' directory2. Activate the plugin through the 'Plugins' menu in WordPress3. Go to Settings -> Yandex WM Page4. Press link "Popup window". In new windows you should allow to access to your yandex profile.5. DO NOT press "Get token" in popup window, just copy code to plugin settings page and Press submit.6. If necessary choose your blog hostname.== Changelog === 0.1.2 =* NEW: Information in dashboard updates after 60 minutes* FIX: If token was canceled, plugin will prompt for new* FIX: Error with cURL Forward in safe mode.= 0.1.1 =* NEW: Information about quoting index* NEW: Number of crawled and indexed pages* NEW: Information about indexation and verification* FIX: Virus information would not be showed, if site is not indexed.= 0.1 =* NEW: Everything
